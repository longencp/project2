Endpoints:

GET: https://project2.cory-longenecker.me:8443/

Shows all of the contents of the database (names, session numbers, balance, and concerns)

GET: https://project2.cory-longenecker.me:8443/campers/:session_num

Shows all of the campers who are registered for that session using the "session_num" parameter

POST: https://project2.cory-longenecker.me:8443/camper

Allows the client to register a camper using a jason file like the one below:

{
  "name":"Cory",
  "session_num":1,
  "balance":2,
  "concerns":"none"
  }

PATCH: https://project2.cory-longenecker.me:8443/camper/:name

Updates the balance of of a specific camper using the "name" parameter

DELETE: https://project2.cory-longenecker.me:8443/camper/:name

Deletes a specific camper using the "name" parameter
